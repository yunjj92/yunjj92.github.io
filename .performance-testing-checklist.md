## ðŸ” Troubleshooting

### Build Fails

```bash
# Check Ruby/Jekyll installation
ruby --version
bundle --version

# Reinstall dependencies
bundle install

# Clear cache and rebuild
bundle exec jekyll clean
bundle exec jekyll build
```

### CSS Not Loading

1. Check file exists: `_site/assets/css/homepage-animations.css`
2. Check path in HTML source
3. Verify custom_css front matter in index.html
4. Check head.html includes custom CSS logic

### Images Not Lazy Loading

1. Check browser support (all modern browsers)
2. Verify `loading="lazy"` attribute in source
3. Check if images are below fold
4. Test in incognito mode (disable extensions)

### Performance Score Lower Than Expected

1. Verify Tailwind CDN is the bottleneck
2. Check Network tab for blocking resources
3. Use Coverage tab to find unused CSS
4. Implement local Tailwind build
5. Enable compression (if self-hosting)
